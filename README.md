## Installation
Install node.js
```bash
node --version
v16.15.0
```
main.ts > app.module.ts > app.controller.ts

```bash
$ yarn install
```

## Running the app

```bash
# development (watch mode)
$ yarn run dev
# app will be run on http://localhost:3000

# production mode
$ yarn run prod
```
